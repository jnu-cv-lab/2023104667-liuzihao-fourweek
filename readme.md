**实验内容**

直接下采样：观察混叠现象

统一 σ 高斯滤波 + 下采样：抗混叠

人脸对比实验：验证人脸 / 背景应使用不同 σ

**依赖**

**结论**

直接下采样混叠严重；统一滤波可抗混叠但丢失细节；自适应下采样能在保留细节的同时抑制混叠；人脸区域应使用小 σ，背景使用大 σ。

**思考题**：
对人脸做 4 倍下采样时，人脸和背景应该用相同 σ 吗？为什么？

不应该用相同 σ。人脸区域包含五官、纹理等高频细节，需要小 σ以保留清晰特征；背景多为平滑低频区域，需要大 σ充分抗混叠。若全局统一 σ，要么人脸模糊，要么背景出现混叠锯齿。



自适应下采样相比统一下采样好在哪里？

自适应能根据图像梯度自动分配滤波强度：高梯度区域少模糊保细节，低梯度区域多模糊防混叠，在画质和抗混叠之间达到更优平衡，整体效果更自然。



为什么棋盘格、chirp 图比人脸更能看出自适应效果？

棋盘格和 chirp 是高频突出、结构规则的合成图，混叠和细节差异非常明显；而人脸是自然图像，高低频过渡柔和，视觉差异不如测试图直观。



梯度在自适应下采样中起什么作用？

梯度用于判断图像局部复杂度：梯度大 = 高频细节区，梯度小 = 低频平滑区，以此动态调整 σ，实现分区自适应处理。
